# SE_Social_App
Study Buddy Finder is a social-based application designed to help students find classmates to study with. The app allows users to create study sessions for specific courses and invite other students to join.

Students can browse available study sessions, view details such as subject, time, and location, and join study groups that match their schedule. This platform encourages collaboration and helps students stay organized with their academic activities.

The application follows the CRUD (Create, Read, Update, Delete) model to manage study sessions and user information.
